## 📖 Описание проекта

Ads Service — это backend-приложение для сервиса объявлений.
Пользователи могут регистрироваться, авторизовываться, создавать объявления,
загружать изображения, оставлять комментарии и управлять своим профилем.

Проект реализован в рамках учебного задания и состоит из четырёх этапов.
## 🧩 Функциональность

### 👤 Пользователи
- Регистрация пользователей
- Авторизация
- Просмотр и обновление профиля
- Загрузка и обновление аватара
- ### 📢 Объявления
- Создание объявлений
- Загрузка изображения объявления
- Просмотр всех объявлений
- Просмотр одного объявления
- Обновление и удаление объявления
- Получение объявлений текущего пользователя

### 💬 Комментарии
- Просмотр комментариев к объявлению
- Добавление комментария
- Обновление комментария
- Удаление комментария
- ### 🖼 Работа с изображениями
- Загрузка изображений
- Хранение изображений в файловой системе
- Получение изображений по HTTP
- Отображение изображений в объявлениях и профиле пользователя

## ▶ Запуск проекта

###  Требования
- Java 17+
- PostgreSQL
- Maven

---
#### Запуск приложения
mvn spring-boot:run


#### Приложение будет доступно по адресу:

http://localhost:8080
📑 Swagger (OpenAPI)

#### Swagger UI доступен по адресу:

http://localhost:8080/swagger-ui/index.html

#### 🔐 Авторизация в Swagger

Используется Basic Authentication.

Зарегистрировать пользователя через /register

Нажать Authorize

Ввести:

username — email пользователя

password — пароль
